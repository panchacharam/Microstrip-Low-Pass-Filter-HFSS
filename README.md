# Microstrip Low Pass Filter Design using HFSS

## Overview

This project presents the design and simulation of a Microstrip Low Pass Filter (LPF) using ANSYS HFSS 2025 R2 Student Version. The filter is designed to allow low-frequency signals to pass while attenuating higher-frequency components.

## Software Used

- ANSYS HFSS 2025 R2 Student

## Filter Specifications

| Parameter | Value |
|------------|------------|
| Filter Type | Microstrip Low Pass Filter |
| Technology | Microstrip Transmission Line |
| Cutoff Frequency | ≈ 2.7 GHz |
| Passband | DC – 2.7 GHz |
| Stopband | Above 3 GHz |
| Simulation Tool | ANSYS HFSS 2025 R2 Student |
